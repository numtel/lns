# Late Night Sketches

[Old site on Wayback Machine](https://web.archive.org/web/20250207082901/http://latenightsketches.com/)

## Running locally

Requires Node.js installed

```sh
$ git clone https://github.com/numtel/lns.git
$ cd lns
# No NPM install required!

# Build into ./dist/
$ npm run build

# Serve (does not watch for changes, must restart)
$ npm run serve

# Build and serve
$ npm run dev
```
